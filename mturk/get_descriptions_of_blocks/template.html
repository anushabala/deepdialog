Instructions

Look at the image below, which shows an arrangement of black blocks on a white grid. 
Write a sequence of commands that can tell a second person how to place blocks on a grid to get this arrangement.
Details:

  • Write a list of commands.
  • Separate each command with the "Enter" key.
  • Each command should describe where in the grid to place a block or group of blocks.
  • You can choose what order the blocks are placed in.
  • The commands should be clear enough that someone who starts from an empty grid and follows your commands should be able to reconstruct the arrangment shown.
